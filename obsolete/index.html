The following ZIP archives are ready TDS installations. Unpack them in the TDS tree you want.

  • base-tds.zip (CTAN:macros/latex/base/)
  • cyrillic-tds.zip (CTAN:macros/latex/required/cyrillic/)
  • graphics-tds.zip (CTAN:macros/latex/required/graphics/)
  • tools-tds.zip (CTAN:macros/latex/required/tools/)
  • amslatex-tds.zip (CTAN:macros/latex/required/amslatex/)
  • psnfss-tds.zip (CTAN:macros/latex/required/psnfss/)
  • babel-tds.zip (CTAN:macros/latex/required/babel/)
  • src-tds.zip (sources of latex-tds)

The zip files so far merged in one file:

  • latex-tds-tds.zip

The documentation files are regenerated:

  • Hyperref features are added (bookmarks, links, ...).
  • The PDF files are post-processed to decrease the file size (without features of PDF-1.5).
